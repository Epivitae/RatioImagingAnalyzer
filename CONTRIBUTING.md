# Contributing to RIA

Thank you for considering contributing to Ratio Imaging Analyzer (RIA)! We welcome contributions from the community, including bug reports, feature suggestions, and code improvements.

## How to Contribute

### 1. Reporting Bugs
If you find a bug, please open an issue in this repository. Include:
- A description of the bug.
- Steps to reproduce it.
- Your operating system and Python version.
- Screenshots (if applicable).

### 2. Suggesting Features
We welcome ideas! Please open an issue and label it as `enhancement`. Describe the feature and why it would be useful for ratio imaging analysis.

### 3. Submitting Code (Pull Requests)
1. **Fork** the repository.
2. Create a new branch for your feature: `git checkout -b feature/my-new-feature`.
3. Make your changes.
4. **Run Tests**: Ensure all tests pass by running `pytest` locally.
5. Commit your changes: `git commit -m 'Add some feature'`.
6. Push to the branch: `git push origin feature/my-new-feature`.
7. Open a **Pull Request** to the `main` branch.

## Coding Standards
- Please follow PEP 8 style guidelines where possible.
- Add comments to explain complex logic, especially in scientific calculation modules.
- If adding a new calculation function, please add a corresponding test case in `tests/`.

Thank you for your support!
